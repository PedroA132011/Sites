O que é HTML?

HTML ou (Hypertext MArkup Language) é uma linguagem de "marcação", e pode ser considerado o corpo do site como o texto, imagens, links e etc...


O que é CSS?

CSS, ou Cascading Style Sheets, é uma linguagem usada para estilizar páginas web.
